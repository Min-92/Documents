# Vim

Vi 에서 개선된 텍스트 에디터

## 일반모드 key

esc를 눌러서 일반모드 진입

- h j k l 커서이동
- x X 삭제
- e w b $ ^ gg G 이동
- dd dw 3dw 5dd 삭제
- i I o O a A 입력모드로 진입
- yy 복사
- p 붙여넣기

## 명령모드

: 일반모드,비주얼모드에서 : 를 눌러서 진입

- :w 저장
- :q 종료
- :q! 저장하지않고 종료
- :wq! 저장하고 종료

## 검색하기

/입력 후 단어 입력 엔터  
/ hello  

- n N 으로 탐색

## 비주얼 모드

v V ctrl+v 를 눌러서 진입  

- v 현재 커서부터 선택
- V 줄단위 선택
- ctrl+v 블럭단위 선택
